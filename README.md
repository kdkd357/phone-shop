# phone-shop
Introduction: This website will display some popular mobile phones for users to place orders, facilitating their online shopping. It provides a search function to enable users to directly find the products they need. Basic shopping cart and payment functions are also available. There is also a contact function to facilitate communication between users and developers for website improvement. A discount section has been developed to directly present the benefits to users.
